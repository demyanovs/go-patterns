# Structural Patterns
Patterns deal with the organization and composition of code to ensure flexibility and scalability.

## 1. Adapter
Helps incompatible interfaces work together.

## 2. Facade
Provides a simplified interface to a complex subsystem.

## 3. Decorator
Dynamically adds behavior or responsibilities to an object.

## 4. Composite
Organizes objects into tree structures to represent whole-part hierarchies.
